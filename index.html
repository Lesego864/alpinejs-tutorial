Creating components using x-data, x-text and x-on:click directives.

A bit of theory:
x-data directive defines a chunk of HTML as an Alpine component and provides the reactive data for that component to reference. x-text sets the text content of an element to the result of a given expression. x-on:click directive executes some javascript when a specific element is clicked.
